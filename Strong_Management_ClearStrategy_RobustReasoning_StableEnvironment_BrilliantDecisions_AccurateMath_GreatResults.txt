### **1. Simulating a Geometric Brownian Motion (GBM) for Asset Prices**
```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>

double generate_random_normal() {
    double u1 = ((double) rand() / RAND_MAX);
    double u2 = ((double) rand() / RAND_MAX);
    return sqrt(-2 * log(u1)) * cos(2 * M_PI * u2);
}

void simulate_gbm(double S0, double mu, double sigma, double T, int steps) {
    double dt = T / steps;
    double St = S0;

    for (int i = 0; i < steps; i++) {
        double dWt = sqrt(dt) * generate_random_normal();
        St = St * exp((mu - 0.5 * sigma * sigma) * dt + sigma * dWt);
        printf("Time %f: Price %f\n", i * dt, St);
    }
}

int main() {
    simulate_gbm(100.0, 0.05, 0.2, 1.0, 100);
    return 0;
}
```
**Purpose:** Simulates continuous-time asset price movements, critical for modeling arbitrage opportunities.

### **2. Risk-Neutral Valuation of a European Call Option**
```c
#include <stdio.h>
#include <math.h>

double risk_neutral_price(double S, double K, double T, double r, double sigma) {
    double d1 = (log(S / K) + (r + 0.5 * sigma * sigma) * T) / (sigma * sqrt(T));
    double d2 = d1 - sigma * sqrt(T);
    return S * 0.5 * (1 + erf(d1 / sqrt(2))) - K * exp(-r * T) * 0.5 * (1 + erf(d2 / sqrt(2)));
}

int main() {
    double S = 100, K = 100, T = 1, r = 0.05, sigma = 0.2;
    printf("European Call Option Price: %f\n", risk_neutral_price(S, K, T, r, sigma));
    return 0;
}
```
**Purpose:** Calculates the arbitrage-free price of an option under the Black-Scholes framework.

### **3. Detecting Arbitrage in a Risk-Free Portfolio**
```c
#include <stdio.h>

int detect_arbitrage(double current_price, double future_price, double r, double T) {
    double no_arbitrage_price = current_price * exp(r * T);
    if (future_price > no_arbitrage_price) return 1;  // Arbitrage exists
    return 0;  // No arbitrage
}

int main() {
    double current_price = 100.0, future_price = 110.0, r = 0.05, T = 1.0;
    if (detect_arbitrage(current_price, future_price, r, T)) {
        printf("Arbitrage detected!\n");
    } else {
        printf("No arbitrage exists.\n");
    }
    return 0;
}
```
**Purpose:** Determines if arbitrage opportunities exist based on no-arbitrage pricing.

```c
#include <math.h>

double binomial_option(double S, double K, double T, double r, double sigma, int steps) {
    double dt = T / steps;
    double u = exp(sigma * sqrt(dt));
    double d = 1 / u;
    double p = (exp(r * dt) - d) / (u - d);
    double prices[steps + 1], values[steps + 1];

        prices[i] = S * pow(u, steps - i) * pow(d, i);

        for (int i = 0; i <= t; i++) {
            values[i] = (p * values[i] + (1 - p) * values[i + 1]) * exp(-r * dt);
        }
    }
    return values[0];
}
```

Martingale methods are foundational in modern financial modeling, particularly for pricing derivatives, risk management, and arbitrage detection. These methods leverage the martingale property—a stochastic process whose future expected value equals its current value under a specific measure. Here’s an advanced and comprehensive overview of martingale methods:

---

### **1. Martingale Definition in Financial Context**
A stochastic process \((M_t)\) is a martingale under a probability measure \(\mathbb{Q}\) if:

\[
\mathbb{E}[M_{t+s} | \mathcal{F}_t] = M_t, \quad \forall s \geq 0.
\]

In financial modeling:
- **Risk-Neutral Measure (\(\mathbb{Q}\)):** Under this measure, discounted asset prices are martingales. This ensures no arbitrage and allows for consistent pricing across all derivative instruments.

---

### **2. Key Applications in Financial Modeling**
#### **a) Derivative Pricing**
The price of a derivative under a risk-neutral measure is given by:
\[
V_t = \mathbb{E}^\mathbb{Q}\left[\frac{1}{B_T} \cdot \text{Payoff} | \mathcal{F}_t\right],
\]
where \(B_T\) is the discount factor. This relationship uses the martingale property of discounted price processes.

#### **b) Fundamental Theorem of Asset Pricing (FTAP)**
FTAP states:
- **No Arbitrage:** A market is arbitrage-free if there exists a risk-neutral measure \(\mathbb{Q}\).
- **Completeness:** If every contingent claim can be replicated, \(\mathbb{Q}\) is unique.

#### **c) Hedging via Replication**
Martingale methods ensure that hedging strategies align with the stochastic behavior of asset prices, minimizing risk under \(\mathbb{Q}\).

---

### **3. Advanced Techniques**
#### **a) Change of Measure (Girsanov’s Theorem)**
- Changes the probability measure from the real-world measure (\(\mathbb{P}\)) to a risk-neutral measure (\(\mathbb{Q}\)).
- This transforms a drifted Brownian motion under \(\mathbb{P}\) into a driftless Brownian motion under \(\mathbb{Q}\), simplifying pricing.

\[
dS_t = rS_t dt + \sigma S_t dW_t^\mathbb{Q},
\]
where \(r\) is the risk-free rate.

#### **b) Martingale Representation Theorem**
In financial modeling, any martingale under \(\mathbb{Q}\) can be expressed as:
\[
M_t = M_0 + \int_0^t \phi_s dW_s^\mathbb{Q},
\]
where \(\phi_s\) represents a predictable process, such as a trading strategy.

---

### **4. Advanced Concepts**
#### **a) Numeraire and Change of Numeraire**
- **Numeraire:** A benchmark asset (e.g., bond, stock) used to normalize prices.
- Changing the numeraire simplifies calculations, particularly for exotic options like quanto and Asian options.

Example:
\[
V_t = \mathbb{E}^\mathbb{Q^N}\left[\frac{Payoff}{N_T} \Big| \mathcal{F}_t\right] \cdot N_t,
\]
where \(N_t\) is the numeraire process and \(\mathbb{Q^N}\) is the equivalent martingale measure.

#### **b) Stochastic Discount Factor (SDF)**
The SDF \(\xi_t = \frac{B_0}{B_t}\) ensures that discounted prices under \(\mathbb{P}\) behave as martingales:
\[
P_t = \mathbb{E}^\mathbb{P}[\xi_t \cdot P_T | \mathcal{F}_t].
\]

---

### **5. Examples of Martingale Models in Finance**
#### **a) Black-Scholes Model**
The price of a European option is determined by the discounted expectation of its payoff under \(\mathbb{Q}\):
\[
C(S_t, t) = \mathbb{E}^\mathbb{Q}\left[e^{-r(T-t)} \max(S_T - K, 0) | S_t\right].
\]

#### **b) Heath-Jarrow-Morton (HJM) Framework**
- Models the evolution of forward rates using martingale techniques.
- Assumes that forward rates under \(\mathbb{Q}\) are driven by Brownian motions.

#### **c) LIBOR Market Model (LMM)**
- Uses the martingale property to model forward LIBOR rates for pricing interest rate derivatives.

```c
double binomial_option_pricing(double S0, double K, double r, double u, double d, int N) {
    double q = (exp(r) - d) / (u - d);  // Risk-neutral probability
    double prices[N + 1], values[N + 1];
    
    // Terminal stock prices
    for (int i = 0; i <= N; i++) {
```

### **2. Jump-Diffusion Process for Asset Price Simulation**  
**Purpose**: Simulate asset prices in a jump-diffusion market using Merton's model.

```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <time.h>

// Jump-diffusion parameters
double generate_jump(double lambda, double mean, double sigma) {
    double jump = 0.0;
    if ((double)rand() / RAND_MAX < lambda) { // Check for jump event
        jump = mean + sigma * ((double)rand() / RAND_MAX - 0.5);
    }
    return jump;
}

void simulate_jump_diffusion(double S0, double mu, double sigma, double lambda, double jump_mean, double jump_sigma, int T, int steps) {
    double dt = (double)T / steps;
    double S = S0;
    printf("Time\tAsset Price\n");

    for (int i = 0; i < steps; i++) {
        double dW = sqrt(dt) * ((double)rand() / RAND_MAX - 0.5); // Random walk
        double jump = generate_jump(lambda, jump_mean, jump_sigma);
        S = S * exp((mu - 0.5 * sigma * sigma) * dt + sigma * dW + jump);
        printf("%.2f\t%.5f\n", i * dt, S);
    }
}

int main() {
    srand(time(0));
    double S0 = 100.0;      // Initial asset price
    double mu = 0.05;       // Drift
    double sigma = 0.2;     // Volatility
    double lambda = 0.1;    // Jump intensity
    double jump_mean = 0.02, jump_sigma = 0.05;
    int T = 1;              // Time horizon in years
    int steps = 100;        // Number of time steps

    simulate_jump_diffusion(S0, mu, sigma, lambda, jump_mean, jump_sigma, T, steps);
    return 0;
}
```

**Reasoning**: Jump-diffusion models provide a realistic representation of asset prices, accounting for market shocks and ensuring robust planning for economic goals.

### **3. Optimal Portfolio Allocation in Jump-Diffusion Markets**  
**Purpose**: Maximize utility considering jumps in asset returns.

```c
#include <stdio.h>
#include <math.h>

// Expected utility calculation with jumps
double expected_utility(double w, double mu, double sigma, double lambda, double jump_mean, double jump_sigma) {
    double continuous_return = mu * w - 0.5 * sigma * sigma * w * w;
    double jump_return = lambda * (jump_mean * w - 0.5 * jump_sigma * jump_sigma * w * w);
    return continuous_return + jump_return;
}

void optimize_portfolio(double mu, double sigma, double lambda, double jump_mean, double jump_sigma) {
    double w, max_utility = -1e9, best_allocation = 0.0;
    for (w = 0.0; w <= 1.0; w += 0.01) {
        double utility = expected_utility(w, mu, sigma, lambda, jump_mean, jump_sigma);
        if (utility > max_utility) {
            max_utility = utility;
            best_allocation = w;
        }
    }
    printf("Optimal allocation: %.2f%% of wealth\n", best_allocation * 100);
    printf("Maximized utility: %.5f\n", max_utility);
}

int main() {
    double mu = 0.08;       // Expected return
    double sigma = 0.25;    // Volatility
    double lambda = 0.1;    // Jump intensity
    double jump_mean = 0.03, jump_sigma = 0.05;

    optimize_portfolio(mu, sigma, lambda, jump_mean, jump_sigma);
    return 0;
}
```

**Reasoning**: This example determines the optimal investment proportion in a risky asset by considering both continuous and jump risks, providing a clear decision-making framework for ambitious goals.

### **4. Dynamic Consumption-Investment Problem**  
**Purpose**: Solve the dual problem of consumption and investment optimization under jump-diffusion dynamics.

```c
#include <stdio.h>
#include <math.h>

// Optimal consumption-investment allocation
void optimize_consumption_investment(double wealth, double risk_free_rate, double risky_mu, double risky_sigma, double lambda, double jump_mean, double jump_sigma) {
    double consumption, investment, max_utility = -1e9;
    double optimal_c = 0.0, optimal_i = 0.0;

    for (consumption = 0.1; consumption <= wealth; consumption += 0.1) {
        for (investment = 0.0; investment <= (wealth - consumption); investment += 0.1) {
            double remaining_wealth = wealth - consumption - investment;
            double risky_return = risky_mu * investment - 0.5 * risky_sigma * risky_sigma * investment * investment;
            double jump_return = lambda * (jump_mean * investment - 0.5 * jump_sigma * jump_sigma * investment * investment);
            double utility = log(consumption) + log(remaining_wealth) + risky_return + jump_return;

            if (utility > max_utility) {
                max_utility = utility;
                optimal_c = consumption;
                optimal_i = investment;
            }
        }
    }

    printf("Optimal consumption: %.2f, Optimal investment: %.2f\n", optimal_c, optimal_i);
    printf("Maximized utility: %.5f\n", max_utility);
}

int main() {
    double wealth = 100.0;
    double risk_free_rate = 0.02;
    double risky_mu = 0.08;
    double risky_sigma = 0.2;
    double lambda = 0.1;
    double jump_mean = 0.03, jump_sigma = 0.05;

    optimize_consumption_investment(wealth, risk_free_rate, risky_mu, risky_sigma, lambda, jump_mean, jump_sigma);
    return 0;
}
```

**Reasoning**: This model ensures comprehensive allocation across consumption, risk-free, and risky assets, delivering actionable insights for wealth maximization.
